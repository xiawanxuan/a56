# 光伏材料实验室 - IV曲线分析系统

**Photovoltaic IV Curve Analyzer v1.0**

光伏材料实验室专用的 Python Qt6 多文件桌面 GUI 分析软件

## 快速开始

### 环境要求
- Python 3.9 或更高版本
- Windows 10/11（已适配）

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 启动程序
```bash
python main.py
```
或者直接双击：启动.bat（Windows）

## 快捷键

| 快捷键 | 功能 |
| --- | --- |
| Ctrl+O | 打开IV数据文件 |
| Ctrl+Shift+O | 文件夹批量导入 |
| Ctrl+S | 保存图像 |
| Ctrl+E | 批量导出 |
| Ctrl+Shift+C | 校准参数配置 |
| Ctrl+U | 单位转换工具 |
| F5 | 重新计算所有参数 |
| Delete | 移除已选曲线 |

## 技术栈

- **GUI**: PyQt6 (Qt6)
- **科学计算**: NumPy, SciPy (Savitzky-Golay平滑, brentq求解器, curve_fit)
- **绘图**: Matplotlib (QtAgg后端)
- **数据处理**: Pandas, OpenPyXL
- **存储**: SQLite3 (标准库)
- **架构**: 模块化分层解耦设计
- **主题**: GitHub风格现代扁平化
